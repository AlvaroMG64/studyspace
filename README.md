# StudySpace

Aplicación web para la gestión de reservas de mesas en bibliotecas.

## Tecnologías
- PHP
- MySQL
- TailwindCSS

## Funcionalidades
- Registro e inicio de sesión
- Roles (admin / usuario)
- Crear, editar y eliminar reservas
- Control de solapamientos
- Panel de administración con filtros y paginación

## Instalación
1. Importar `studyspace.sql` en MySQL
2. Configurar `config/db.php`
3. Ejecutar en servidor local (XAMPP, etc.)

## Usuario administrador
Crear un usuario y cambiar su rol a `admin` en la base de datos.
